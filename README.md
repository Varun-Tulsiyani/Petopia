# Petopia

### CSIT128 - Introduction to Web Technology

An interactive web project to explore pet-related content and user engagement.

---
### Overview:

**Petopia** is a web application built as part of the CSIT128 course. It combines HTML, CSS, and JavaScript to present
a static website focused on pet care, adoption, or pet community content. This project can serve as a foundation for
showcasing pet profiles, providing pet information, and engaging users with interactive UI elements.

### Tech Stack:

- HTML
- CSS
- JavaScript (ExpressJS)
- MySQL
